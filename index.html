每个本科生寝室都有一个百兆的端口，本教程的目的就是告诉大家如何通过这个端口来为寝室四个人的电脑手机或其他设备提供上网服务。

几个方案

一般正常家用路由器是一个 WAN 口 + 四个 LAN 口，方案一二均够用。
网线根据寝室结构和布线方式自行购买，注意长度，建议到寝室实测后再买。
选择困难症患者建议方案一，简单粗暴快捷有效。

方案一

需要：一台路由器，一个 IP，至少一根网线
注意：所有人的电脑不必进行任何设置

  1. 一个同学申请 IP，获取相关信息（详见 如何申请IP）
  2. 登陆路由器管理界面
  3. 根据申请所得信息设置路由器
  4. 网线连接墙上端口和路由器 WAN 口（一般来说就是那个长得与众不同的口）

在此方案下，所有设备均能通过路由器来无线上网，有网线接口的电脑可以选择用网线连接路由器来上网。

方案二

理论上来说会比方案一快一点。。。

需要：一台交换机，一台路由器，一至四个 IP，至少两根网线

  1. 一个或一个以上同学申请 IP，获取相关信息（详见 如何申请IP）
  2. 一根网线连接墙上端口和交换机，另一根网线连接交换机和路由器 WAN 口
  3. 用某同学A的信息设置路由器，同方案一

此时效果同方案一，所有设备都可以通过路由器上网，无需设置电脑。

除同学A以外申请了 IP 的同学，另外可以选择通过交换机上网。

  1. 用网线连接电脑和交换机（不是路由器）
  2. 在电脑上设置相关信息，参见 NIMO官方指南 第8步以后

方案三（不常见方案，未亲测）

此方案为无线路由器作无线交换机使用，通过关闭 DHCP 服务

需要：一台路由器

  1. 注意：如果四个人都申请了 IP 都希望有线连接路由器上网，则需要5个 LAN 口的路由器
  2. 网线连接墙上端口和路由器某一个 LAN 口（注意是 LAN 口不是 WAN 口，不同于方案一二）
  3. 路由器设置时，关闭 DHCP 服务
  4. 无线网络正常连接

电脑可以通过网线连接路由器，但是此时要在电脑上设置相关信息，同方案二直连交换机的电脑设置，参见 NIMO官方指南 第8步以后

方案四（基本不可能）

需要：一台交换机

放弃在寝室使用无线网络，所有电脑通过网线连接交换机上网。每个人申请一个IP，具体设置同方案二最后两条。

申请开通网络

注意：

  • 申请开网的页面需要通过交大校内的网络访问，建议连上名为SJTU的Wi-Fi（教学楼、图书馆、餐厅各处均有覆盖，寝室楼内没有）后再申请，或者通过挂交大的代理来进入页面
  • 如果有同学想在家里挂交大代理来申请，请务必确认你已经获知以下必需信息：Jaccount 帐号，寝室楼号和寝室号，手机号（最好是上海号码）

具体也可以参考 NIMO官方指南 1~6步。

  1. 查看本机 mac 地址。详见下文 查看Mac 或下拉查看
  2. 到NIMO网站上填写申请 戳我

查看MAC地址

这里最重要的一点就是不要把有线和无线搞错。

Windows

方法一：

  1. 打开运行（快捷键为徽标键(win) + r ）
  2. 输入cmd，点击确定或按下回车
  3. 出现一个黑乎乎的框，不要害怕，输入 getmac -v 并按下回车
  4. 查看以太网（也可能叫本地连接）对应的物理地址（注意不要和无线网络连接搞错，否则到时候又要更改信息，比较麻烦）。一般形如C8-0A-A9-6B-3B-57，有英文有数字，十二位，每两位由减号分隔
  5. 这个就是你需要的MAC地址了，把它记录下来

方法二：

  1. 找到右下角任务栏中网络图标并右键，点击打开网络和共享中心（或从控制面板进入），win7 则打开网络连接
  2. 找到以太网（本地连接）的图标并右键，选择状态
  3. 点击详细信息
  4. 其中物理地址一项就是MAC地址，形如C8-0A-A9-6B-3B-57

其他：

  • 将方法一中的 getmac -v 替换为 ipconfig/all 或 ipconfig -all 并回车，然后查看以太网（本地连接）对应的物理地址
  • 在方法二第2步中右键选择属性，将鼠标移至第一个长条形的框中，不要点击，此时会显示MAC地址

MacOS

如果有同学使用USB外接网卡来使用有线上网，则查看 网络设置-高级-以太网，下面有一个以太网ID就是了。或者也可以打开终端输入 ifconfig 查看。

Linux

ifconfig 或 ip addr

路由器设置

  1. 在Wi-Fi列表中找到你的路由器并连接
  2. 打开浏览器，输入路由器背面的路由器管理界面的IP（一般为192.168.1.1，有的TP-LINK路由器是 tplogin.cn 比较奇葩）
  3. 输入路由器背面默认的用户名和密码（一般为admin和admin，如果不能登陆可以尝试戳路由器背后的复位键来恢复出厂设置）
  4. 进入网络参数-WAN口设置，连接类型改为静态IP，然后在下面填入申请所得信息（IP地址，子网掩码，网关，DNS）
  5. 进入网络参数-MAC地址克隆，将MAC地址改为申请时用的MAC地址
  6. 在无线设置中开启无线功能，并设置 Wi-Fi 名称及密码
  7. 建议修改登陆口令及密码
  8. 设置完成，重启路由器
