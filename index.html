  • 一些废话
  • 申请 IP
    参见该指南1~6步 http://nimo.sjtu.edu.cn/aton/index.php?g=Home&m=Index&a=newapply& 方案一需要一个同学申请，方案二1~4个任选
    • 查看本机 mac 地址
    • 到网管部官网填写申请
    • 等待开通
  • 几个方案
    一般正常家用路由器是一个 WAN 口 + 四个 LAN 口，方案一二均够用。
    • 一台路由器，一个 IP，至少一根网线
      • 一个同学申请 IP，获取相关信息
      • 所有人的电脑不必进行任何设置
      • 登陆路由器管理界面
      • 根据申请所得信息填写 IP，子网掩码，网关，DNS，更改 mac 地址
      • 网线连接墙上端口和路由器 WAN 口
      • 所有设备均能通过路由器来无线上网，有网线接口的电脑可以选择用网线连接路由器来上网
    • 一台交换机，一台路由器，一至四个 IP，至少两根网线
      理论上来说会比方案一快一点。。。
      • 一个及以上同学申请 IP
      • 一根网线连接墙上端口和交换机，另一根网线连接交换机和路由器 WAN 口
      • 用某同学A的信息设置路由器，具体步骤同方案一
      • 所有同学都可以通过路由器上网，具体步骤同方案一，无需设置电脑，网线连接路由器或无线上网
      • 除A以外申请了 IP 的同学，另外可以选择通过交换机上网
        • 网线连接电脑和交换机
        • 电脑相关信息设置
          参见 http://nimo.sjtu.edu.cn/aton/index.php?g=Home&m=Index&a=applyguide& 第8步以后
    • 一台路由器（不常见方案，未亲测）
      此方案为无线路由器作无线交换机使用，通过关闭 DHCP 服务
      • 注意：如果四个人都申请了 IP 都希望有线连接路由器上网，则需要5个 LAN 口的路由器
      • 网线连接墙上端口和路由器某一个 LAN 口（注意是 LAN 口不是 WAN 口，不同于方案一二）
      • 路由器设置时，关闭 DHCP 服务
      • 无线网络正常连接
      • 电脑可以通过网线连接路由器，但是此时要设置相关信息，同方案二直连交换机的电脑设置
        参见 http://nimo.sjtu.edu.cn/aton/index.php?g=Home&m=Index&a=applyguide& 第8步以后
